技术文档 ​

记录技术规范、架构设计、最佳实践等内容。

前端开发规范 ​

JavaScript/TypeScript ​

  • 代码风格：ESLint 配置、Prettier 格式化
  • 类型系统：TypeScript 类型定义、类型体操
  • 模块管理：ES 模块、CommonJS、Tree Shaking

架构设计 ​

  • 组件设计：单一职责、可复用组件、组件通信
  • 状态管理：Vuex、Pinia、Redux、Zustand
  • 路由管理：Vue Router、React Router

性能优化 ​

  • 代码分割：动态导入、webpack 优化
  • 懒加载：图片懒加载、路由懒加载
  • 打包优化：构建体积、加载速度优化

后端架构 ​

Node.js ​

  • 事件循环：异步编程原理、Promise、async/await
  • 内存管理：V8 内存机制、内存泄漏排查
  • 进程管理：Cluster、PM2、Docker 部署

数据库设计 ​

  • SQL：MySQL、PostgreSQL 查询优化
  • NoSQL：MongoDB、Redis 数据模型设计
  • 数据迁移：数据库版本控制、迁移脚本

系统架构 ​

  • 微服务：服务拆分、API 网关、负载均衡
  • 消息队列：RabbitMQ、Kafka、Redis Stream
  • 缓存策略：Redis、Memcached、本地缓存

开发流程 ​

Git 工作流程 ​

  • 分支策略：Git Flow、Trunk Based Development
  • 提交规范：Conventional Commits、Changelog
  • 代码审查：PR 流程、Code Review 规范

部署流程 ​

  • CI/CD：GitHub Actions、GitLab CI
  • 环境管理：开发、测试、生产环境配置
  • 监控日志：ELK、Prometheus、Grafana
