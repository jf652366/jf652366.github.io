# 学习笔记

这里记录了我在学习和开发过程中积累的技术知识、实践经验和问题解决方案。

## 技术分类

### 前端开发
- **Vue.js**：组件开发、状态管理、路由配置
- **React**：Hook 用法、性能优化、生态工具
- **TypeScript**：类型系统、高级特性、最佳实践

### 后端开发
- **Node.js**：Express、Koa、NestJS 框架使用
- **数据库**：MySQL、MongoDB、Redis 数据操作
- **API 设计**：RESTful API、GraphQL、Websocket

### 系统架构
- **架构设计**：微服务、模块化架构、系统设计
- **性能优化**：代码优化、数据库优化、缓存策略
- **安全防护**：XSS、CSRF、SQL注入防护
