# AI 相关

关注人工智能领域的最新动态、工具和学习资源，包括大模型应用、AI 开发等。

## 大模型

### 对话模型
- **Claude**：Anthropic 开发的对话模型
- **GPT**：OpenAI 的生成式预训练模型
- **Gemini**：Google 的多模态模型

### 图像模型
- **DALL-E**：图像生成模型
- **Midjourney**：AI 绘画工具
- **Stable Diffusion**：开源图像生成模型

## 开发工具

### AI 编程助手
- **Copilot**：GitHub 的代码生成工具
- **CodeWhisperer**：Amazon 的编程助手
- **Claude Code**：Anthropic 的代码生成工具

### 模型服务
- **OpenAI API**：提供各种大模型的访问接口
- **Claude API**：Anthropic 的 API 服务
- **Hugging Face**：开源模型库和推理服务

## 学习资源

### 在线课程
- **吴恩达 AI 课程**：DeepLearning.AI
- **斯坦福 CS229**：机器学习经典课程
- **Fast.ai**：实践导向的深度学习课程

### 学习平台
- **Kaggle**：数据科学竞赛和学习平台
- **Coursera**：各种 AI 相关课程
- **edX**：MIT、哈佛等名校课程

## 应用场景

### 开发辅助
- **代码生成**：快速生成代码片段和函数
- **代码优化**：自动优化代码性能和可读性
- **Bug 修复**：辅助定位和修复代码问题

### 内容创作
- **文本生成**：文章、邮件、文档写作
- **图像创作**：AI 绘画、设计素材生成
- **多媒体处理**：音频、视频内容处理
