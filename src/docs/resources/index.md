# 资源导航

收集常用的开发资源、学习网站和实用工具链接。

## 学习平台

### 编程学习
- **GitHub**：开源项目和代码仓库
- **Stack Overflow**：编程问题解答社区
- **LeetCode**：算法和数据结构练习
- **牛客网**：面试和算法训练平台

### 前端学习
- **MDN Web Docs**：Web 开发文档
- **DEV Community**：开发者社区和教程
- **W3C**：Web 标准和规范
- **caniuse**：浏览器兼容性查询

### 后端学习
- **Node.js 中文网**：Node.js 文档和教程
- **MongoDB 中文社区**：MongoDB 文档和资源
- **Redis 中文社区**：Redis 文档和最佳实践

## 开源项目

### 前端框架
- **Vue.js**：渐进式 JavaScript 框架
- **React**：用于构建用户界面的库
- **Angular**：前端开发平台
- **Svelte**：编译时的 JavaScript 框架

### 后端框架
- **Express**：Node.js Web 应用框架
- **Koa**：Node.js 下一代 Web 框架
- **NestJS**：渐进式 Node.js 框架
- **Fastify**：高性能 Node.js Web 框架

- **Lodash**：JavaScript 工具库
